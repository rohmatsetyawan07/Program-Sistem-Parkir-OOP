# Program-Sistem-Parkir-OOP
Program Sistem Parkir sederhana yang dibuat menggunakan bahasa C++ dengan konsep Object Oriented Programming (OOP). Program ini mencatat waktu masuk dan keluar kendaraan secara otomatis serta menghitung biaya parkir berdasarkan durasi.
Deskripsi Program
Sistem ini mensimulasikan proses parkir kendaraan, mulai dari:

Kendaraan masuk
Pencatatan waktu otomatis
Kendaraan keluar
Perhitungan durasi parkir
Penentuan biaya parkir
Menampilkan struk parkir
Program dijalankan melalui terminal / command line.

Fitur Utama
Input plat nomor dan jenis kendaraan
Waktu masuk dan keluar tercatat otomatis
Gratis 10 menit pertama
Perhitungan biaya parkir per jam
Batas maksimal biaya parkir
Menampilkan struk parkir ke layar
Menggunakan konsep Class & Method (OOP)
